# NepaliRuby

Write Ruby Code in Nepali Language.

नेपाली भाषामा रुबी कोड लेख्नुहोस । 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nepali_ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nepali_ruby
